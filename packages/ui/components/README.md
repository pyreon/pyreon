# @pyreon/ui-components

Production-ready UI components for Pyreon — built on rocketstyle with multi-state styling, dark mode, and accessibility.

## Install

```bash
bun add @pyreon/ui-components
```

## Usage

```tsx
import { Button, Card, Badge, Alert } from '@pyreon/ui-components'

<Card>
  <Badge state="success" size="sm">Active</Badge>
  <Button state="primary" size="md">Click me</Button>
  <Alert state="info">Welcome!</Alert>
</Card>
```
