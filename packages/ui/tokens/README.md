# @pyreon/ui-tokens

Design tokens for the Pyreon UI component library.

## Install

```bash
bun add @pyreon/ui-tokens
```

## Usage

```ts
import { colors, spacing, fontSize, radii, shadows } from '@pyreon/ui-tokens'

colors.primary[500]  // '#3b82f6'
spacing[4]           // 16
fontSize.md          // 16
radii.lg             // 8
shadows.md           // '0 4px 6px -1px ...'
```
